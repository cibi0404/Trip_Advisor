# Trip_Advisor

TripAdvisor Clone
A front-end clone of TripAdvisor built with plain HTML and CSS. No frameworks, no libraries — just fundamentals.

project/

Navbar
Logo + brand name

Headline and subtext
Search bar with icon, input, and search button
Popular Destinations
4-column CSS Grid
Cards with image, city name, country, star rating, review count, and badge
Hover lift effect on cards
Top Experiences

Cards with image, title, star rating, and price with strikethrough original price
Footer
4-column grid with link columns (Discover, Explore, Business, Get the App)
Copyright bar at the bottom
